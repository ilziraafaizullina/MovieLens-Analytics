# MovieLens Analytics

## Описание проекта

В рамках проекта проведен анализ пользовательских оценок фильмов на основе данных MovieLens.

Цель — выявить закономерности в оценках, поведении пользователей и характеристиках фильмов, а также представить результаты в виде аналитического отчета.

---

## Задачи проекта

- Анализ распределения пользовательских оценок  
- Выявление популярных и высоко оцененных фильмов  
- Анализ динамики оценок по времени  
- Сегментация пользователей по уровню удовлетворенности  
- Поиск противоречивых фильмов (с высокой вариативностью оценок)  

---

## Данные

В проекте использовались:

- данные о фильмах  
- пользовательские оценки  
- теги  
- дополнительные данные, полученные с помощью парсинга  

Объем данных:
- ~1000 записей  
- ~100 пользователей  
- ~1000 фильмов  

---

## Архитектура решения

Реализован модуль на Python с использованием объектно-ориентированного подхода:

- `Movies` — работа с данными о фильмах  
- `Ratings` — обработка пользовательских оценок  
- `Tags` — работа с тегами  
- `Links` — обработка ссылок  

Каждый класс инкапсулирует логику обработки соответствующего типа данных.

---

## Тестирование

Для проверки корректности работы реализованы тесты:

- проверка типов возвращаемых данных  
- проверка корректности структуры данных  
- проверка сортировки результатов

## Аналитика

В ходе анализа были получены следующие результаты:

### 1. Топ фильмов по годам
- определены наиболее популярные фильмы в разные периоды  

---

### 2. Динамика оценок
- исследована временная динамика средней пользовательской оценки  
- выявлены изменения предпочтений пользователей

### 3. Сегментация пользователей

- пользовательские оценки позволяют выявлять предпочтения аудитории

- высокая дисперсия оценок может указывать на неоднозначное восприятие фильма  
- сегментация пользователей помогает лучше понять поведение аудитории
